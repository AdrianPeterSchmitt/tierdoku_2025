## 📋 Nützliche Befehle:

### Server starten:
```powershell
php -S localhost:8000 -t public
```

### Server stoppen:
Drücke `Ctrl+C` im Terminal

### Code formatieren:
```powershell
vendor/bin/pint
```

### Static Analysis:
```powershell
vendor/bin/phpstan analyse app
```

### Tests ausführen:
```powershell
vendor/bin/phpunit
```

### TailwindCSS watch (automatisch rebuild):
```powershell
npm run watch
```
